=org-draw= (=org-excalidraw.el=) lets you create, open, export, and display Excalidraw diagrams directly from Org mode.

* Requirements

To work as expected, the following requirements must be fulfilled.

** Open Drawings

To open and edit Excalidraw files, you can:

1. Install an Excalidraw app in Google Chrome
2. Make Excalidraw the default application to open files with the =.excalidraw= extension
3. Use the resource opener of your desktop environment to verify:

   - For Linux, run ~xdg-open /path/to/example.excalidraw~
   - For Mac, run ~open /path/to/example.excalidraw~

** Create Thumbnails

SVG thumbnails are created for exporting and displaying. To achieve this, one of the following tools must be set up correctly.

- [preferred] [[https://github.com/yuzutech/kroki-cli][yuzutech/kroki-cli]] — install via binary; see the upstream manual for setup
- [[https://github.com/Timmmm/excalidraw_export][Timmmm/excalidraw_export]] — install via ~npm install -g excalidraw_export~

* Installation

** async-installer

[[https://github.com/zawatton21/async-installer][async-installer]] — a small, non-blocking package manager for Emacs. Pure Elisp syntax — no macro DSL required.

#+begin_src emacs-lisp
(async-installer-git-add "https://github.com/zawatton21/org-draw.git"
                            :branch "master"
                            :main "org-draw.el")
#+end_src

** straight.el

#+begin_src emacs-lisp
(straight-use-package
 '(org-draw :type git :host github :repo "zawatton21/org-draw"))
#+end_src

** elpaca

#+begin_src emacs-lisp
(elpaca (org-draw :host github :repo "zawatton21/org-draw"))
#+end_src

** use-package (with straight or elpaca)

#+begin_src emacs-lisp
;; with straight
(use-package org-draw
  :straight (org-draw :type git :host github :repo "zawatton21/org-draw"))

;; with elpaca
(use-package org-draw
  :ensure (:host github :repo "zawatton21/org-draw"))
#+end_src

** Manual

Clone this repository and add it to your =load-path=:

#+begin_src emacs-lisp
(add-to-list 'load-path "/path/to/org-draw")
(require 'org-draw)
#+end_src

* Usage

- Treat the =excalidraw= link type the same way Org mode handles other link types
- Command =org-excalidraw-create-drawing= creates a blank drawing
- Customization =org-excalidraw-default-directory= specifies where new diagrams are stored
